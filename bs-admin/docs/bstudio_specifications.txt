■module/editor
 function regist

 セーブする前にバージョンをチェックし、変わっている場合はコンテンツIDを更新する。
　updateNodeファンクションで行う。
　バージョン削除があった場合、旧ファイルが存在する可能性があるがその場合は「だれかが更新している」ダイアログが表示される→しょうがない。

■module/resource
 function saveName

 ファイル名を変更した場合、通常はコンテンツIDは変更しない（画像ファイルはバージョン間で共有）
　但し、拡張子を変えた場合はファイルの拡張子も同時に変更するのでバージョンが違っている場合は、別ファイルを作成しコンテンツIDを変更する。
　サムネイルファイルも同時に更新する。
